## 二、数据源详细说明

### 1. Space-Track satcat API (主要数据源)

**访问地址**: `https://www.space-track.org/basicspacedata/query/class/satcat/format/json`

**认证方式**: 需要注册账号，使用用户名密码登录获取cookie

**API文档**: https://www.space-track.org/documentation#satcat

**实际返回字段** (2026年3月验证):

```
INTLDES, NORAD_CAT_ID, OBJECT_TYPE, SATNAME, COUNTRY, LAUNCH, SITE, DECAY,
PERIOD, INCLINATION, APOGEE, PERIGEE, COMMENT, COMMENTCODE, RCSVALUE, RCS_SIZE,
FILE, LAUNCH_YEAR, LAUNCH_NUM, LAUNCH_PIECE, CURRENT, OBJECT_NAME, OBJECT_ID, OBJECT_NUMBER
```

**字段映射关系**:

| Space-Track字段         | 系统字段        | 说明               | 备注  |
| --------------------- | ----------- | ---------------- | --- |
| NORAD_CAT_ID          | noradId     | NORAD编号          |     |
| OBJECT_NAME / SATNAME | name        | 卫星名称             |     |
| OBJECT_ID / INTLDES   | objectId    | 国际编号             |     |
| OBJECT_TYPE           | objectType  | 对象类型             |     |
| COUNTRY               | countryCode | 国家/所有者           |     |
| LAUNCH                | launchDate  | 发射日期             |     |
| SITE                  | launchSite  | 发射场代码            |     |
| DECAY                 | decayDate   | 衰减日期             |     |
| PERIOD                | period      | 轨道周期             |     |
| INCLINATION           | inclination | 轨道倾角             |     |
| APOGEE                | apogee      | 远地点              |     |
| PERIGEE               | perigee     | 近地点              |     |
| RCS_SIZE              | rcs         | 雷达截面分类           |     |
| CURRENT               | status      | 在轨状态 (Y→+, 其他→D) |     |

**不包含的字段**:

- LAUNCH_VEHICLE (运载工具) - satcat表中没有此字段
- ALT_NAMES (别名) - satcat表中没有此字段
- STD_MAG (标准星等) - satcat表中没有此字段

### 2. CelesTrak TLE API (TLE数据源)

**访问地址**: `https://celestrak.org/NORAD/elements/gp.php?GROUP=active&FORMAT=tle`

**认证方式**: 公开免费，无需认证

**数据格式**: 三行TLE格式 (名称 + Line1 + Line2)

**可解析的轨道参数**:

| TLE位置         | 字段           | 说明             | 备注  |
| ------------- | ------------ | -------------- | --- |
| Line1 位置19-32 | epoch        | 历元时间           |     |
| Line2 位置9-16  | inclination  | 轨道倾角           |     |
| Line2 位置18-25 | raan         | 升交点赤经          |     |
| Line2 位置27-33 | eccentricity | 偏心率 (需加"0."前缀) |     |
| Line2 位置35-42 | argOfPerigee | 近地点幅角          |     |
| Line2 位置53-63 | meanMotion   | 平均运动           |     |

### 3. CelesTrak satcat API (备用数据源)

**访问地址**: `https://celestrak.org/satcat/records.php`

**状态**: 当前API可能已变更或不可用，系统已改用Space-Track作为主要数据源

## 四、字段值说明

### status (在轨状态)

来源: Space-Track satcat `CURRENT` 字段

| 值   | 含义     | 转换规则          | 备注  |
| --- | ------ | ------------- | --- |
| +   | 在轨运行   | CURRENT = "Y" |     |
| D   | 已衰减/再入 | CURRENT ≠ "Y" |     |

### rcs (雷达截面)

来源: Space-Track satcat `RCS_SIZE` 字段

| 值      | 含义        | 备注  |
| ------ | --------- | --- |
| LARGE  | > 100 m²  |     |
| MEDIUM | 10-100 m² |     |
| SMALL  | < 10 m²   |     |

### objectType (对象类型)

来源: Space-Track satcat `OBJECT_TYPE` 字段

| 值           | 含义          | 备注  |
| ----------- | ----------- | --- |
| PAYLOAD     | 有效载荷 (卫星本体) |     |
| ROCKET BODY | 火箭箭体        |     |
| DEBRIS      | 碎片          |     |

### launchSite (发射场代码)

来源: Space-Track satcat `SITE` 字段

| 代码     | 名称         | 备注  |

| TTMTTR | 拜科努尔航天发射场  |     |
| JSC    | 酒泉卫星发射中心   |     |
| XSC    | 西昌卫星发射中心   |     |

## 六、外部数据源

### 1. N2YO API

**访问地址**: https://www.n2yo.com/api/

**认证方式**: 需要API Key

**可获取字段**: mission, user, purpose, contractor, launchMass, lifetime, platform 等

### 2. Gunter's Space Page

**访问地址**: https://space.skyrocket.de/

**获取方式**: 需要爬虫

**可获取字段**: 运载工具、质量、尺寸、寿命、平台、载荷等详细技术参数

### 3. ESA DISCOS

**访问地址**: https://discosweb.esoc.esa.int/

**认证方式**: 需要申请访问

**可获取字段**: 质量、长度、直径、翼展、横截面、形状代码等

## 八、后续优化建议

### 可快速实现的优化

1. **运载工具信息**: 可从N2YO API获取，或从Gunter's爬取
2. **别名**: 可从N2YO或Gunter's获取
3. **标准星等**: 部分卫星可从CelesTrak获取(如API可用)

### 需要额外工作的优化

1. **任务/用途信息**: 集成N2YO API
2. **物理参数**: 申请ESA DISCOS访问权限
3. **技术细节**: 建立Gunter's爬虫

### 数据同步策略

1. TLE数据: 每日更新 (CelesTrak)
2. 元数据: 每周更新 (Space-Track)
3. 任务信息: 每月更新 (N2YO)
